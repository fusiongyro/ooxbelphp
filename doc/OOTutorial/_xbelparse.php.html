This is an attempt to illustrate some principals of object-orientation.

What you'll find within:

  • A SAX-based generic XML parser class, SAXParser
  • A simple XML structure displaying parser, TagDisplay
  • A SAXParser derivative for parsing XBEL documents, XBELParser
  • A set of XBEL structural classes, XBELNode, XBELFolder and XBELBookmark
  • A simple XBEL Visitor-pattern implementation, XBELVisitor
  • An example use of XBELVisitor, URLGatherer, which gathers up URLs from under all the nodes in the XBEL document.
